# Explainable IDS

An Explainable Intrusion Detection System (IDS) with a modern dashboard.

## Features
- Traffic spike detection
- Severity-based alerts
- Explainable reasoning
- FastAPI backend
- Next.js frontend

## Tech Stack
- Python (FastAPI)
- Next.js
- Tailwind CSS
- Recharts

### Backend

cd backend
uvicorn main:app --reload
